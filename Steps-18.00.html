Creating refreshed date column and Adding Arrow Indicators 18
  • Creating refreshed date column 
  1. Click on the 3 dots (...) to the right of the DateTable on the Data Pane. Select New measure
  2. On the measures field type this formula RefreshDate = NOW(), this will give you a Date and time. Format the date to general. Your new column will appear in the DateTable  


  • Creating refreshed date card
  1. Copy the Days of Inventory Card by CTRL C and paste CTRL V 
  2. Drag and Drop the RefreshedDate measure from DateTable to the Fields in the Build pane
  • Visual Format
Change Title to -  Refreshed
Callout Value: Font = DIN 12



Adding Arrow Indicators
  • Inventory turnover rate arrow indicator
  1. Click on an empty space on the canvas, and from the visualizations pane select Visual Card
    1. Drag the  ..Arrow Inv turn measure to the fields value box or drag to the visual card
    2. Fill up figures provided below for the necessary fields
  2. Format Pane: Visual 
  3.  Size and position: Height=37, Width=51, Horizontal=117, Vertical=253
  4.  Title=Off
  5.  Effects: Background=Off                
  6. Category Label = off
Visual:
  1.   Callout Value: Font=DIN 15
  2.    Font color=fx (click on the function tab and fill up values as displayed below)
  3. Green color = #49AC28; Red color = #E66C37; if blank = White
