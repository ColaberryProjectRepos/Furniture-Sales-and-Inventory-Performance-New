Creating Waterfall and line and stacked column charts 20
  • Creating Profit by Sub-Category Waterfall visual
  1. Click on an empty space on the canvas, and from the visualizations pane select Waterfall Chart
  2. Fill up figures provided below for the necessary fields
Build Visual Pane:
  • X axis = Sub-Category, Product Name
  • Y axis= Sum of Profit
Format Visual Pane:

Visual:
  • Size and Position: Height=341, Width=333, Horizontal=177, Vertical=378
  • Background =On; Transparency = 59%
  • Visual border =On; Color= White #FFFFFF
  •  Title=On, Text=Profit by Sub-Category, Font=DIN (13), Text color=#73361C, Background color=#E8D166, Horizontal alignment=Center
  • X-axis: Values = On, Font=Segoe UI 9; Color=#73361C; Max height 25%; Title=On, DIN 12; Color=#73361C
  • Y-axis: Range: Auto, Values=On, Font=Segoe UI 9; Color=#73361C; Title=On, DIN 12; Color=#73361C
  • Legend: on, Position =Top left; Text= Seqoe UI 10; Color =#73361C; Title=off


  • Creating Quantity Sold and Days of Inv by segment Line and Stacked Column Chart
  1. Click on an empty space on the canvas, and from the visualizations pane select Line and Stacked Column Chart (You can also copy and paste the sales, inventory and stockout analysis bar chart that you already created previously and just change chart type and fill out all the necessary fields as specified below)
  2. Fill up figures provided below for the necessary fields
Build Visual Pane:
  • X axis = Segment
  • Column Y axis= Total Quantity Sold
  • Line y-axis = Days of Inventory 
Format Visual Pane:

Visual:
  • Size and Position: Height=341, Width=369, Horizontal=510, Vertical=378
  • Background =On; Transparency = 59%
  • Visual border =On; Color= White #FFFFFF
  •  Title=On, Text=Quantity Sold and Days of Inv by Segment, Font=DIN (13), Text color=#73361C, Background color=#E8D166, Horizontal alignment=Center
  • X-axis: Values = On, Font=Segoe UI 9; Color=#73361C; Max height 25%; Title=On, DIN 12; Color=#73361C
  • Y-axis: Range: Auto, Values=On, Font=Segoe UI 9; Color=#73361C; Title=On, DIN 12; Color=#73361C
  • Secondary y-axis: Values=on; Font Segoe UI 9; Color=#73361C
  • Legend: on,  Text= Seqoe UI 10; Color =#73361C; Title=off
  • Columns: Categories =All; Color =#73361C;
  • Lines: Series=All; Line Color #12239E;
