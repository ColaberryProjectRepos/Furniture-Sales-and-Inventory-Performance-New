Creating Scatter Chart 21
  • Creating Scatter Chart - Inv Turnover Rate, Days of Inv and Sum of Qty by Sub-Category
  1. Click on an empty space on the canvas, and from the visualizations pane select Scatter chart
  2. Fill up figures provided below for the necessary fields

Build Visual Pane:
  • Values  = Category Hierachy - sub-Category and Product Name
  • X axis= Inventory Turnover Rate
  • Y-axis = Days of Inventory 
  • Size = Sum of Quantity
Format Visual Pane:

Visual:
  • Size and Position: Height=339, Width=399, Horizontal=880, Vertical=378
  • Background =On; Transparency = 59%
  • Visual border =On; Color= White #FFFFFF
  •  Title=On, Text=Inv Turnover Rate, Days of Inv and Sum of Qty by Sub-Category, Font=DIN (13), Text color=#73361C, Background color=#E8D166, Horizontal alignment=Center
  • X-axis: Values = On, Font=Segoe UI 9; Color=#73361C;  Title=On, DIN 12; Color=#73361C
  • Y-axis: Range: Auto, Values=On, Font=Segoe UI 9; Color=#73361C; Title=On, DIN 12; Color=#73361C
  • Secondary y-axis: Values=on; Font Segoe UI 9; Color=#73361C
  • Legend: on,  Text= Seqoe UI 10; Color =#73361C; Title=on; Title Text = Total Sales 
  • Markers: Categories=All; Color- Color by category =on
  • Category Label: on; Values - Font=Seguoe UI 9; Color=#73361C
