Adding a Canvas Background and Create the Larger Visuals/Charts 19
  • Canvas Background
Click on an empty space on the canvas and on the Format pane click the Canvas Background and upload an image from your local computer. Set transparency to 59%.

  • Creating Total Quantity Sold by Year plus Forecast Line chart
  1. Click on an empty space on the canvas, and from the visualizations pane select Line Chart
  2. Fill up figures provided below for the necessary fields
Build Visual Pane:
  • X axis = Year, MonthName
  • Column y axis= Total Quantity Sold
Format Visual Pane:

Visual:
  • Size and Position: Height=309, Width=332, Horizontal=178, Vertical=69
  • Background =On; Transparency = 59%
  • Visual border =On; Color= White #FFFFFF
  •  Title=On, Text=Total Quantity Sold by Year plus Forecast, Font=DIN (13), Text color=#73361C, Background color=#E8D166, Horizontal alignment=Center
  • X-axis: Values = On, Font=Segoe UI 9; Color=#73361C; Title=On, DIN 12; Color=#73361C
  • Y-axis: Values=On, Font=Segoe UI 9; Color=#73361C; Title=On, DIN 12; Color=#73361C
  • Lines: Color=#73361C
  • Forecast: On; Options Units= Points; Forecast Length=5; Seasonality=1; Confidence level=95%; Forecast line - Color #73361C; Confidence Shadow=On; Match line color=On



  • Creating Sales, Inventory and Stockout analysis Clustered column Chart
  1. Click on an empty space on the canvas, and from the visualizations pane select Clustered Column Chart
  2. Fill up figures provided below for the necessary fields
Build Visual Pane:
  • X axis = Sub-Category
  • Column y axis= 
-HighSales_LowInventory_AvgSales; 
-LowSales_HighInventory_AvgSales; 
-ProductsAtRiskOfStockout; 
-ExcessInventoryCount
  • Tooltips - from measures table drop the Sum of Inventory Level to tooltip 
Format Visual Pane:

Visual:
  • Size and Position: Height=309, Width=369, Horizontal=510, Vertical=69
  • Background =On; Transparency = 59%
  • Visual border =On; Color= White #FFFFFF
  •  Title=On, Text=Sales, Inventory and Stockout analysis, Font=DIN (13), Text color=#73361C, Background color=#E8D166, Horizontal alignment=Center
  • X-axis: Values = On, Font=Segoe UI 9; Color=#73361C; Title=On, DIN 12; Color=#73361C
  • Y-axis: Range: Logarithmic scale =on, Values=On, Font=Segoe UI 9; Color=#73361C; Title=On, DIN 9; Color=#73361C
  • Legend=on, Text= Seqoe UI 10; Color =#73361C
  • GridLines: Horizontal=On
  • Columns Color: Series -HighSales_LowInventory_AvgSales =#73361C; LowSales_HighInventory_AvgSales=#E8D166; ProductsAtRiskOfStockout=#CC754F; ExcessInventoryCount=#AB8B7D




  • Creating Map -Forecast Accuracy by City and State
  1. Click on an empty space on the canvas, and from the visualizations pane select Map
  2. Fill up figures provided below for the necessary fields
Build Visual Pane:
  • Location = CityState
  • Bubble Size = Forecast Accuracy
Format Visual Pane:

Visual:
  • Size and Position: Height=307, Width=399, Horizontal=880, Vertical=71
  • Background =On; Transparency = 59%
  • Visual border =On; Color= White #FFFFFF
  •  Title=On, Text=Forecast Accuracy by City and State Font=DIN (13), Text color=#73361C, Background color=#E8D166, Horizontal alignment=Center
  • Map Settings: Style=Road; Show labels= on; Controls - Autozoom=off
  • Bubbles:  Size - range scale = Auto; size -10; colors - choose conditional format button (fx) - Fill the following on the box that pops up - Format Style - Gradient; Field - Forecast Accuracy; Minimum = lowest value color #73361C; middle value =  #FFFFFF; highest value =A38600 press 'ok' when done
