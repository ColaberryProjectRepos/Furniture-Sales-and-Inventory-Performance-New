Create Measures table and DAX Measures 15
  1. Create a new table to put all your DAX measures -Go to the Home tab  click on  Enter Data on the ribbon and name the blank table  MeasureTable and click Load
  • MeasureTable 




  • Create DAX Measures
  1. Go to Modeling tab on Ribbon 
  2. Click on New Measure 
  3. Fill in all the DAX formulas provided below, one at a time, and when you finish Press Enter or select Commit (checkmark icon) in the formula bar.  
  4. After creating each measure choose the necessary format and decimal for each value (measure)
  5. Save
DAX Formulas:

  • Days of Inventory = 
    DIVIDE(
        AVERAGE('Furniture Inventory'[Inventory Level]),
        AVERAGE('Prediction_Furnituredata'[Quantity]) 
    ) 
  • ExcessInventoryCount = 
CALCULATE(
    COUNTROWS('Furniture Inventory'),
    FILTER(
        'Furniture Inventory',
        'Furniture Inventory'[Inventory Level] > 400
    )
)
  • Forecast Accuracy = 
    1 - ABS( 
        SUM('Prediction_Furnituredata'[Quantity]) - SUM('Prediction_Furnituredata'[prediction])
    ) / SUM('Prediction_Furnituredata'[Quantity])

  • HighSales_LowInventory_AvgSales = 
CALCULATE(
    AVERAGE('Prediction_Furnituredata'[Sales]),
    FILTER(
        'Prediction_Furnituredata',
        'Prediction_Furnituredata'[Sales] > 700 
        && RELATED('Furniture Inventory'[Inventory Level]) < 100
    )
)

  • Arrow HighSales_LowInventory_AvgSales = 
SWITCH(
    TRUE(), 
    [HighSales_LowInventory_AvgSales] < 700, 
    UNICHAR(9660),  // Downward filled arrow (Red)
    [HighSales_LowInventory_AvgSales] > 700, 
    UNICHAR(9650),  // Upward filled arrow (Green)
    UNICHAR(9654)   // Right arrow (Neutral, can be changed as needed)
)
  • Inventory Turnover Rate = 
    DIVIDE(
        SUM('Prediction_Furnituredata'[Quantity]),
        AVERAGE('Furniture Inventory'[Inventory Level])
    )
  • ..Arrow Inv turn = 
SWITCH(
    TRUE(), 
    [Inventory Turnover Rate] > 5, 
    UNICHAR(9660),  // Downward filled arrow (Red)
    [Inventory Turnover Rate] < 5, 
    UNICHAR(9650),  // Upward filled arrow (Green)
    UNICHAR(9654)   // Right arrow (Neutral, can be changed as needed)
)

  • LowSales_HighInventory_AvgSales = 
    CALCULATE(
        AVERAGE('Prediction_Furnituredata'[Sales]),
        FILTER(
            'Prediction_Furnituredata',
            'Prediction_Furnituredata'[Sales] < 700 
            && RELATED('Furniture Inventory'[Inventory Level]) > 400
        )
    )
  • Arrow LowSales_LowInventory_AvgSales = 
    SWITCH(
        TRUE(), 
        [LowSales_HighInventory_AvgSales] < 700, 
        UNICHAR(9660),  // Downward filled arrow (Red)
        [LowSales_HighInventory_AvgSales] > 700, 
        UNICHAR(9650),  // Upward filled arrow (Green)
        UNICHAR(9654)   // Right arrow (Neutral, can be changed as needed)
    )
  • ProductsAtRiskOfStockout = 
    CALCULATE(
        COUNTROWS('Furniture Inventory'),
        FILTER(
            'Furniture Inventory',
            'Furniture Inventory'[Inventory Level] < 100
        )
    )
  • Total Quantity Sold = 
    SUM('Prediction_Furnituredata'[Quantity])
  • Total Sales = 
    SUM('Prediction_Furnituredata'[Sales])
