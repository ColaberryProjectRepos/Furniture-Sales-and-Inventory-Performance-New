Create Year Slicer, KPI visuals, Arrow Indicators and Refresh Date 17

  • Year Slicer
  1. Click on an empty space on the canvas, and from the Build pane select Slider visual
  2. On the Data pane from DateTable drag the Year to the fields value box
  3. Fill up figures provided below for the necessary fields
Format - visual
Size and style -  Height=96, Width=175, Position - Horizontal=1, Vertical=69
Background - on, Color = #E8D166; Transparency = 59%
Title=On, Text=Year, Font=DIN(14), Text color=#73361C, Background color=#E8D166, Horizontal alignment=Center
Slicer header = off
           
Visual: Values, Font=Segoe UI (9), Font color=#73361C
            Slider=On, Slider Color=#73361C



  • Creating Total Sales KPI
Click on an empty space on the canvas, and from the Build pane select Card visual
  1. On the build pane drag and drop the Total Sales measure on the fields format it to decimal number 0.
visual Format
size and position - Height =62, width=177; Horizontal =0, vertical=166
Background = on, color #E8D166, transparency =59%
Title on = Total Sales, Font = DIN 13, Text color #73361C, Background=E8D166
Horizontal alignment= center
callout value (number)- Font =Din 20, color =#73361C
Category Label =off


  •  Creating Inventory Turnover rate KPI
  1. Copy Total Total Sales KPI by pressing CTRL C and paste it by CTRL V
  2. On the build pane drag and drop the Inventory Turnover Rate measure from the MeasureTable on the fields (delete the Total Sales field). Format the value to 0 decimals.
visual Format
size and position - Height =62, width=177; Horizontal =0, vertical=166
Background = on, color #E8D166, transparency =59%
Title on = Inv turnover rate, Font = DIN 13, Text color #73361C, Background=E8D166
Horizontal alignment= center
callout value (number)- Font =Din 20, color =#73361C
Category Label =off

  • Creating Product at stockout Risk KPI
  1. Copy Total Total Sales KPI by pressing CTRL C and paste it by CTRL V change the visual to a KPI card by selecting the KPI visual on the Build pane
  2. On the build pane drag and drop the ProductsAtRiskofStockout measure from the MeasureTable on the Value field and Sub-Category to the Trend Axis field. Format the value to whole number.
Visual Format
Change Title to Prod at Stockout Risk
callout value: Font =Din 20, color =#73361C

  • Creating Excess Inv Count KPI
  1. Copy Prod at Stockout Risk KPI by pressing CTRL C and paste it by CTRL V 
  2. On the build pane drag and drop the ExcessInventoryCount measure from the MeasureTable on the Value field and Sub-Category to the Trend Axis field. Format the value to whole number.
Visual Format
Change Title to -  Excess Inv Count
(because you copied and pasted other visual all the formats are already applied)


  • Creating HighSales_LowInv Avg KPI
  1. Copy Prod at Stockout Risk KPI by pressing CTRL C and paste it by CTRL V 
  2. On the build pane drag and drop the HighSales_LowInventory_AvgSales measure from the MeasureTable on the Value field and Sub-Category to the Trend Axis field. Format the value to Currency and 0 decimals.
Visual Format
Change Title to -  HighSales_LowInv Avg
