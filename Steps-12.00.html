Importing Data to Power BI Desktop 12

Expected Dashboard



Screenshot of the CSV file


  1. Download the Furniture and Sales Inventory file by clicking the red icon above
  2. Open Power BI Desktop
  3.  Click on Get data ribbon.
  4.  Select Text/CSV and then click connect
  5. Open saved file, choose  Furniture data and Inventory CSV files and click on Open
  6. A preview of the data will be displayed.
  7. Ensure data is correct and clean as necessary using the Power Query Editor tools
  8. Once you have completed any transformation click close and apply
  9. Upload the 2nd file Inventory CSV the same way as the Furniture data.
